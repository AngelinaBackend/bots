# Бот для Модерации Чатов

Этот бот для Telegram создан с использованием библиотеки `aiogram` и предназначен для управления и модерации чатов. Бот позволяет администраторам выдавать предупреждения, банить и разбанивать пользователей, а также автоматически удаляет сообщения от забаненных пользователей.

## Описание

Бот помогает администраторам чатов управлять поведением пользователей, предоставляя инструменты для предупреждения нарушителей, блокировки и разблокировки пользователей. Он реагирует на команды `/warn`, `/unwarn`, `/ban` и `/unban`, позволяя поддерживать порядок в чате.

## Технологии в проекте

- **Python 3.7+**
- **aiogram** — библиотека для работы с Telegram API
- **Asyncio** — для асинхронного выполнения задач
- **InlineKeyboardMarkup** и **InlineKeyboardButton** — для создания клавиатуры с кнопками

## Характерные особенности проекта

- **Управление предупреждениями:** Бот позволяет администраторам выдавать предупреждения пользователям, которые могут привести к бану после достижения лимита.
- **Бан и разбан пользователей:** Администраторы могут банить и разбанивать пользователей, а также удалять сообщения от забаненных пользователей.
- **Автоматическая отправка приветственного сообщения:** Бот отправляет приветственное сообщение при добавлении в чат и предоставляет инструкции по использованию команд.

## Установка и запуск

1. **Клонируйте репозиторий:**

   ```bash
   git clone https://github.com/AngelinaBackend/bots.git
   cd tgbotAdmin
   
2. Установите необходимые пакеты:

     pip install aiogram
   
3. Создайте конфигурационный файл:

   - Создайте файл с именем config.py в корневой директории проекта.
   - Добавьте следующее содержимое с вашим токеном бота:

         token = 'ВАШ_ТОКЕН_БОТА_ЗДЕСЬ'
     
4. Запустите бота:

     python main.py
   
## Команды и функции

- /warn — Предупреждает пользователя. Используется в ответ на сообщение пользователя.
- /unwarn — Снимает предупреждение с пользователя. Используется в ответ на сообщение пользователя.
- /ban — Банит пользователя. Используется в ответ на сообщение пользователя.
- /unban — Разбанивает пользователя. Используется в ответ на сообщение пользователя.
- /start — Отправляет приветственное сообщение и кнопку для добавления бота в чат.

## Хранение данных

- Бот использует внутренние структуры данных для хранения списка забаненных пользователей и предупреждений.

## Логирование

- Логирование настроено на вывод информационных сообщений и предупреждений в консоль.
